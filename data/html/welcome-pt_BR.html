Contato Comercial

Para tratar assuntos comerciais, fazer um primeiro contato para uma proposta nova, e agendar uma visita, entre em contato com nossa divisão comercial <comercial @ stoq.com.br>.

Os telefones para contato são (16) 3376-0125 e (16) 9165-0344.

Contato Técnico

Para contactar a equipe do projeto Stoq, ou para encaminhar dúvidas, sugestões ou soluções utilize as listas de discussão.

Para tratar de assuntos técnicos como desenvolvimento, arquitetura e usabilidade, é recomendado falar com o responsável pelo projeto, Christian Reis <comercial-stoq @ async.com.br>. Comentários sobre o site ou alguma das páginas Web que o compõem devem também ser enviados para este mesmo e-mail.

Caso possua uma conexão à internet disponível, você pode entrar em contato pelo IRC.

No menu aplicações, internet, execute o programa XChat IRC. Escolha um Nick name para identificar-se, e conecte-se ao Freenode. Após isso, você deverá entrar no canal #stoq.

Por fim, deseje um bom dia para os desenvolveres que estiverem disponíveis no canal, faça alguns elogios e tire suas dúvidas :-).
