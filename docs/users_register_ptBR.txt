Cadastro de Usuários
====================

Para o cadastro de novos usuários no Administrativo, clique no botão
*Adicionar* na tela principal. Para facilitar a tarefa, um assistente de
cadastro será exibido. 

Inicialmente, escolha entre as opções: *Pessoa Física* e *Pessoa Jurídica*
de acordo com o perfil do novo usuário. A escolha entre uma dessas opções é
sempre necessária.

.. image:: ../../images/admin/new_user.png
   :align: center

Caso o telefone informado já faça parte do cadastro de algum usuário do
Stoq, o sistema informará a duplicação.

Para o cadastro de usário de Pessoa Física, o Admistrativo exibe quatro abas
para preenchimento: *Detalhes do Usuário*, *Identificação*, *Dados de Pessoa
Física*, *Detalhes Gerais*.Para Pessoa Jurídica, a única diferença é a aba
*Dados de Pessoa Jurídica*. Estas abas serão detalhadas a seguir. 

Note que os campos com preenchimento obrigatório estão marcados com      
|notebook|. A principal aba é *Descrição do Usuário*, e caso existam outros
campos obrigatórios em outras abas, estas serão destacadas em vermelho.

Editar Usuário
==============

Para *Editar Usuário*, escolha o usuário a ser editado na lista de usuários.
Altere os dados desejados usando as mesmas abas sugeridas para a criação de
usuários. Para finalizar, a qualquer momento da edição, clique em "Ok".

Alterar senha
=============

Para alterar a senha de um usuário cadastrado, clique no botão 'Alterar
senha'. A tela a seguir será exibida. Informe a senha atual e sugira uma
nova senha para o usuário.

.. image:: ../../images/admin/user_change_pass.png
   :align: center
